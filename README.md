## 📌 About

**AOS-PACMAN** is a Pac-Man game developed in the context of an Operating Systems (AOS) course. The project explores core OS concepts such as process management, concurrency, and inter-process communication through the lens of a classic arcade game.

---

## ✨ Features

- Classic Pac-Man gameplay mechanics
- Ghost AI with basic pathfinding
- Score tracking
- OS-level concepts applied (processes, threads, synchronization)

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- A C/C++ compiler (e.g. `gcc`, `g++`)
- `make` (optional but recommended)

### Installation

```bash
# Clone the repository
git clone https://github.com/mohamed-ali-halloul/AOS-PACMAN.git

# Navigate to the project directory
cd AOS-PACMAN
```

---

## 🎮 Usage

```bash
# Compile the project
make

# Run the game
./pacman
```
